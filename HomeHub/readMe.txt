This is a Arduino Home automation library required to control ESP12 based home automation devices.Contains all the hardware implementation and various advanced task scheduling internally while providing a simple interface to the user.

This revision contains:
-> MQTT Support.
-> Serial Slave handling Support.
-> Wire control over handling DS3231 timer and 32K External Flash control over the module.
-> Contains memory Management functions of DS3231 32K memory.
-> DNS based Hotspot Web-server setup code embedded.
-> Complete Wifi Data management code.
-> Added Json Code capturing from slave serial port.
-> Added OTA feature including OTA check on boot and OTA on demand from cloud.[MQTT command remaining for OTA on demand]
