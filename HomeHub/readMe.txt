This is a Arduini Home automation library required to control ESP12 based home automation devices.Contains all the hardware implementation and various advanced task scheduling internally while providing a simple interfact to the user.

This revision contains:
-> MQTT Support.
-> Serial Slave handling Support.
-> Wire control over handling DS3231 timer and 32K External Flash control over the module.
-> Contains memory Management fucntions of DS3231 32K memory.
-> DNS based Hotspot Webserver setup code embidded.
-> Complete Wifi Data management code.
